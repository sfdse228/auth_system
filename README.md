# 🔐 Модуль авторизации и управления доступом

Консольное приложение с регистрацией, входом, лимитом попыток, аудит-логом и защищённой зоной.

## 📋 Функции

- **Регистрация** — создание нового пользователя
- **Авторизация** — вход с лимитом 3 попыток
- **Блокировка** — при 3 неудачных попытках блокировка на 5 секунд
- **Защищённая зона** — доступ только после успешного входа
- **Логирование** — все действия записываются в лог-файл
